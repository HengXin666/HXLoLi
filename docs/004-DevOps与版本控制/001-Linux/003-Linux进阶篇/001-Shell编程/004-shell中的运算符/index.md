# shell中的运算符
运算符的功能：在脚本中实现各种运算操作
基本语法:
```bash
# 第一种写法
$((表达式))

# 第二种写法
$[表达式]

# 第三种写法
# 注意: 在expr中加减乘除的运算符: + 加 - 减 \* 乘 / 除 % 取余数
`expr m 运算符 n`
# expr与运算式之间要有空格
# m n必须是一个数字，并且数字和运算符之间必须要有空格。
# expr 做计算，只能实现两个操作数的计算
```

示例:

```bash
#!\bin\bash
echo "a + b = $[$1 + $2], a * b = $(($1 * $2))"
```

```bash
[root@localhost shellCode]# sh 05_yunsuanfu.sh 6 6
a + b = 12, a * b = 36
```
