# Linux环境搭建--gcc g++

## 安装

运行安装命令

```Bash
yum install -y gcc gcc-c++
```

安装成功后，试一下gcc和g++是否安装成功

```Bash
gcc -v
g++ -v
```

## 拓展: Kawaii-gcc 让GCC变得可爱

为您的终端安装日语（如果未安装）
```Bash
yum install language-pack-ja language-pack-gnome-ja language-pack-ja-base language-pack-gnome-ja-base
```
