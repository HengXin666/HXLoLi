# Linux日期指令
## date

`date`: 获取当前系统时间

常用参数:

- `-s`: 设置系统时间

`date "+%Y-%m-%d %H:%M:%S"（功能描述：显示年月日时分秒）`

```Shell
[root@hxlinux dev]# date
2024年 01月 05日 星期五 01:13:32 CST
[root@hxlinux dev]# date "+%H:%M"
01:13

[root@hxlinux dev]# date -s "2024-1-4 17:14:08"
2024年 01月 04日 星期四 17:14:08 CST
```
