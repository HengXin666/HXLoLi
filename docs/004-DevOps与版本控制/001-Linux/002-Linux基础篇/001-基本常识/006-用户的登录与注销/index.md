# Linux用户的登录与注销
## 命令
### su [用户名]
切换用户, 从 当前 用户切换到 [用户名] 用户;

如果 `当前用户`权限**高于**`目标用户`, 那么不需要输入密码; 否则需要输入密码.

```Bash
su [name]
```

注: *如果提示用户不存在, 建议把名称写为 英文的小写, 再尝试*

### exit
退出登录(注销)

如果你使用`su`从 root --> heng_xin --> root 的话.

那么 `exit` 会从 root --> heng_xin (回退)

再次 `exit` 才 heng_xin --> root (回退)

再次 `exit` root --> 注销(退出 远程登录)
