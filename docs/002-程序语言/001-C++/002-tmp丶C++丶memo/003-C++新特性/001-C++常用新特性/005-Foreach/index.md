# For each
## 说明
`C++11` 引入了基于范围的迭代写法，拥有了能够写出像 Python 类似的简洁的循环语句。

## 示例

```C++
int main()
{
    vector<int> v1={ 1,2,3,4,5,6 };
    for (auto i : v1)//范围for
    {
    	cout << i << “ ”;
    }
    return 0;
}
```

## 注意

`for (auto& it : 容器)`的 it 是容器的元素的引用, 速度会快一点(只是省略了拷贝操作), 并且 对 `it` 进行修改, 也会**作用**到 容器的元素!

`for (auto it : 容器)`中, 对 `it` 不会对容器的元素有任何影响, 因为它是元素的拷贝.
