英文缩写索引

  1. PC - 个人计算机.[1.1]
  2. WSC - 计算机.[1.1]
  3. CPU - 控制器 和 数据通路 统称: (中央)处理器.[1.4]
  4. ALU - 运算逻辑单元/算数逻辑单元.[1.4/3.1]
  5. DRAM(Dynamic Random Access Memory) - 动态随机存取存储器, 最为常见的系统内存.[1.4]
  6. SARM - 静态随机访问存储器.[1.4]
  7. RAM(Random Access Memory) - 随机存取存储器, 俗称 (运行)内存.[1.4]
  8. ROM(Read-Only Memory) - 只读存储器, 且断电不会丢失数据.[1.4]
  9. CPI(clock cycle per instruction) - 指令性能, 表示执行每条指令所需的时钟周期数的平均值.[1.5]
  10. IPS - 每秒执行了多少条指令.[1.5]
  11. MIPS(million instructions per second) - 每秒百万条指令.[1.5]
  12. PC(Program Counter) - 程序计数器.[2.6]
  13. PLA - 可编程逻辑阵列.[4.3]
  14. LCA - 逻辑单元阵列.[4.3]
  15. ISA - 指令集架构.[4.5]
  16. IR - 指令寄存器.[4.6]
  17. MAR(Microaddress Register) - 微地址寄存器.[4.6]
  18. MPC(Microprogram Counter) - 微程序计数器.[4.6]
  19. MIR(Microinstruction Register) - 微指令寄存器.[4.6]
  20. CISC(Complex Instruction Set Computer) - 复杂指令集计算机.[4.6]
  21. RISC(Reduced Instruction Set Computer) - 精简指令集计算机.[4.6]
