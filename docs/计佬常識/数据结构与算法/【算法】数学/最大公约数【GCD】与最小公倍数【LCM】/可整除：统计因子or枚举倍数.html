统计因子/枚举倍数

给你两个整数数组 nums1nums1nums1 和 nums2nums2nums2，长度分别为 nnn 和 mmm。同时给你一个正整数 kkk。

如果 nums1[i]nums1[i]nums1[i] 可以被 nums2[j]×knums2[j] \times knums2[j]×k 整除，则称数对 (i,j)(i, j)(i,j) 为 优质数对 ( 0≤i≤n−1,0≤j≤m−10 \le i \le n - 1, 0 \le j \le m - 10≤i≤n−1,0≤j≤m−1 )。

返回 优质数对 的总数。

  • 学习链接: 两种方法：统计因子/枚举倍数（Python/Java/C++/Go）

统计因子​

我们知道, 如果 nums1[i]nums1[i]nums1[i] 可以被 nums2[j]×knums2[j] \times knums2[j]×k 整除, 则有 nums1[i] % (nums2[j]×k)==0nums1[i] \ \% \ (nums2[j] \times k) == 0nums1[i] % (nums2[j]×k)==0

即 nums2[j]×knums2[j] \times knums2[j]×k 是 nums1[i]nums1[i]nums1[i] 的因子, 因此我们可以先统计 nums1[i]nums1[i]nums1[i] 因子的个数, 再遍历 nums2[j]nums2[j]nums2[j] 看它是否在 nums1[i]nums1[i]nums1[i] 的因子里面.

  • 我们知道统计一个数因子的算法的时间复杂度是 O(n)O(\sqrt{n})O(n​) 的.

类比判断质数的那个:

  • 如果一个数 ddd 可以 整除 xxx, 那么 d(d∈[1,x])d (d \in [1, x])d(d∈[1,x]) 是 xxx 的一个因子, 但是同时 xd\frac{x}{d}dx​ 也是 xxx 的一个因子, 所以我们在 O(n)O(\sqrt{n})O(n​) 就可以一次找到所有的因子!

  • 特别注意的是: 如果 d×d=xd \times d = xd×d=x 时候, 会重复统计, 因此需要去掉一个!

代码如下, (小优化: d≤x⟺d×d≤xd \le \sqrt{x} \Longleftrightarrow d \times d \le xd≤x​⟺d×d≤x (左右同时平方嘛~))

unordered_map<int, int> cnt;

int x = 114514; // 统计 114514 的因子
for (int d = 1; d * d <= x; d++) {
    if (x % d)
        continue;
    
    cnt[d]++;
    if (d * d < x)
        cnt[x / d]++;
}

总的代码如下: (小优化: a[i] % (b[i] * k) <=> (a[i] / k) % (b[i]), 这样可以减少枚举范围)

class Solution {
public:
    long long numberOfPairs(
        vector<int>& nums1, vector<int>& nums2, int k) {
/*
(a[i] / k) % (b[i]) == 0

枚举 a[i] / k 的 因子
*/
        unordered_map<int, int> cnt;
        for (int it : nums1) {
            if (it % k)
                continue;
            it /= k;
            for (int i = 1; i <= sqrt(it); ++i) {
                if (it % i == 0) {
                    ++cnt[i];
                    if (i * i < it)
                        ++cnt[it / i];
                }
            }
        }
        long long res = 0;
        for (int it : nums2)
            res += cnt.count(it) ? cnt[it] : 0;
        return res;
    }
};

枚举倍数 (调和级数枚举)​

  1. 先去重 nums1nums_1nums1​ 和 nums2nums_2nums2​ 到 cnt1cnt_1cnt1​ 和 cnt2cnt_2cnt2​ 中, 记 nums1nums_1nums1​ 中元素最大值为 uuu.

  2. 枚举 cnt2cnt_2cnt2​ 的元素 iii, 然后枚举 iii 的倍数: i,2i,3i,...i, 2i, 3i, ...i,2i,3i,... 直到 uuu, 然后看看这些值有那些是在 cnt1cnt_1cnt1​ 的, 累加到 sss 中.

class Solution {
public:
    long long numberOfPairs(
        vector<int>& nums1, vector<int>& nums2, int k) {
/*
(a[i] / k) % (b[i]) == 0

b[i] 的倍数, 使其不超过 max(a[i] / k)
*/
        unordered_map<int, int> cnt1, cnt2;
        for (int it : nums1) {
            if (it % k)
                continue;
            ++cnt1[it / k];
        }
        if (!cnt1.size())
            return 0;

        int maxx = 0;
        for (auto [i, _] : cnt1)
            if (i > maxx)
                maxx = i;

        for (int it : nums2)
            ++cnt2[it];

        long long res = 0;
        for (auto [i, v] : cnt2) {
            int s = 0;
            for (int j = i; j <= maxx; j += i) { // 枚举倍数
                s += cnt1.count(j) ? cnt1[j] : 0;
            }
            res += (long long) s * v;
        }

        return res;
    }
};
