判断质数

什么是tm的质数​

  • 质数: 即除了 111 和 它本身它本身它本身, 不能被其他数整除的数.

质数是指 大于1 且只能被 111 和 自身自身自身 整除的 正整数, 1 不是不是不是 质数

暴力求解 O(N)O(N)O(N)​

bool ifFxxk(int m) { // 判断 m 是不是质数
    for (int i = 2; i < m; ++i)
        if (!(m % i))
            return false;
    return m > 1;
}
C++

简单优化 O(N)O(\sqrt{N})O(N​)​

根据上面代码, 对于 121212 和 131313 我们如何判断他们是不是质数:

  1. 如果12 % 2 == 0, 那么12 % 2 * 2 == 0

    • 如果12 % 3 == 0, 那么12 % 3 * 2 == 0

  2. 如果13 % 2 != 0, 那么13 % 2 * 2 != 0

    • 如果13 % 3 == 0, 那么13 % 3 * 2 == 0

显然我们不需要循环这么多次, 因为 m 如果不能被 2 整除, 那么也一定不能被 2 * k 整除.

所以我们最多只需要循环到 sqrt(m) 即可.

故:

bool ifFxxk(int m) { // 判断 m 是不是质数
    int n = sqrt(m);
    for (int i = 2; i <= n; ++i)
        if (!(m % i))
            return false;
    return m > 1;
}
C++

使用 i≤mi \le \sqrt mi≤m​ 等价于 i2≤mi^2 \le mi2≤m

bool ifFxxk(int m) { // 判断 m 是不是质数
    for (int i = 2; i * i <= m; ++i)
        if (!(m % i))
            return false;
    return m > 1;
}
