# 数据库基础知识
## 数据库
### 概念
数据库，英文名称为`Database`，简称`DB`。从字面意思来看就是存储数据的仓库；从专业角度解释为存储在计算机磁盘上的有组织、可共享的大量数据的集合。

### 数据库的类型
数据库分为`关系型`数据库和`非关系型`数据库两大类。
- 常见的关系型数据库有
  - `MySQL`
  - Oracle
  - SQLServer
  - SQLite
  - DB2 等
- 常见的非关系型数据库有
    - `Redis`
    - MongoDB 等。

### 数据库管理系统
数据库管理系统，英文名称为`Database Management System`，简称`DBMS`。主要用于科学组织和存储数据、高效的获取和维护数据。
